Our Journey

2021

WaterIntake Tracker

Our first major app launched to the public. We developed an intuitive hydration tracking solution using Flutter's responsive widget system and local storage capabilities to help users maintain healthy hydration habits.

2022

MathFlash Guru

Expanded into the education sector with an interactive mathematics learning application. We implemented advanced Flutter animations and custom state management to create engaging flashcard experiences for students of all ages.

2023

Moyo Match

Ventured into social connectivity with our dating app featuring video call capabilities. We integrated WebRTC technology and Firebase backend to create seamless real-time communication and secure user matching algorithms.

2024

SabbathGPT

Pioneered AI integration in mobile apps with SabbathGPT, leveraging OpenAI's API with Flutter to develop an intelligent spiritual companion. This project showcased our ability to implement complex natural language processing within a mobile framework.

2025

PenziVibe

Launched our most sophisticated dating platform yet, incorporating facial recognition using TensorFlow and Google ML Kit for enhanced security and better matching. The app features advanced personalization algorithms and real-time messaging capabilities.
